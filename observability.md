# Observability

## Logging
The application logs API requests and errors.

Logs can be collected using CloudWatch or other log aggregation tools.

## Metrics
Important metrics monitored:

- Request count
- Error rate
- Response latency
- Health endpoint status

## Monitoring
Cloud monitoring tools can track application health and performance.

## Alerts
Alerts should be triggered if:

- Error rate exceeds 5%
- Health endpoint fails
- Service latency becomes too high
